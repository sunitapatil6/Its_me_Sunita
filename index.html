Academic Biography

Mrs. Sunita Ramesh Patil is an Assistant Professor in the Department of Computer Engineering at Dr. D. Y. Patil Institute of Technology, Pimpri, Pune. She is currently pursuing her Ph.D. in Computer Science and Engineering from Amity University, Mumbai, focusing on intelligent systems and cognitive skill impairment modeling. She has completed her Master of Engineering from the University of Pune and has over 14 years of rich experience in engineering academics and academic administration. She is an author of SPPU course reference books, a syllabus revision coordinator, and has published over 15 patents and numerous research papers.

Subjects Taught (Semester Details)
3 Featured Courses
Web Engineering and Technology (WE)
Code: WE-401 Semester: VII Credits: 4 Lectures: 3 hrs/week

Course Objectives:

To introduce web engineering methodologies, responsive HTML5/CSS3 page formatting, client-side dynamic scripting (JavaScript/DOM), backend integrations (PHP/Java Servlets), and database systems connectivity.

Course Outcomes (COs):

  • CO1: Formulate structured web documents using HTML5 semantic layouts.
  • CO2: Design responsive styles using media queries and CSS grid structures.
  • CO3: Apply JavaScript to implement interactive client validation and DOM modifications.
  • CO4: Build database-backed web architectures using server-side logic (PHP/Java).

Syllabus Overview:

  • Unit I: Web Engineering Principles and Client-Server Architectures
  • Unit II: HyperText Markup Language 5 (HTML5) & Cascading Style Sheets (CSS3)
  • Unit III: Client-Side Interactivity: JavaScript Syntax, Arrays, Functions, and DOM
  • Unit IV: Dynamic Server-Side Web Content Generation using PHP
  • Unit V: Database Connectivity (MySQL/JDBC) & Session Management
  • Unit VI: Web Services (RESTful APIs), Security, XML, and JSON Protocols

Recommended Reference Books:

  • Textbook: Kogent Learning Solutions Inc., "Web Technologies: HTML, JavaScript, PHP, Java, JSP, XML, AJAX, Black Book", Dreamtech Press.
  • Reference: Thomas Powell, "Web Design: The Complete Reference", Tata McGraw Hill.
Operating System (OS)
Code: OS-302 Semester: V Credits: 3 Lectures: 3 hrs/week

Course Objectives:

To discuss process structures, CPU scheduling algorithms, inter-process synchronization semaphores, deadlock conditions, virtual memory structures, and file system layouts.

Course Outcomes (COs):

  • CO1: Evaluate quantitative computer OS performances via scheduler simulations.
  • CO2: Resolve race issues inside concurrent threads using semaphores.
  • CO3: Analyze page replacement policy behaviors on virtual memory buffers.

Syllabus Overview:

  • Unit I: Overview of OS Structures, Kernel Types, and System Calls
  • Unit II: Process Management & CPU Scheduling (FCFS, SJF, Round Robin)
  • Unit III: Process Synchronization, Critical Section Problem, and Semaphores
  • Unit IV: Deadlocks Detection, Prevention, and Banker's Avoidance Algorithm
  • Unit V: Memory Management, Paging, Segmentation, and Virtual Memory Page Replacements
  • Unit VI: Disk Storage Scheduling and File Directories Layouts

Textbooks & References:

  • Textbook: Silberschatz, Galvin, and Gagne, "Operating System Concepts", Wiley.
  • Reference: William Stallings, "Operating Systems: Internals and Design Principles", Pearson.
Core JAVA Programming
Code: JAVA-203 Semester: III Credits: 4 Lectures: 3 hrs/week

Course Objectives:

To learn object-oriented constructs, inheritance systems, multi-threaded pipelines, exception controls, Java Collections framework, and socket programming.

Course Outcomes (COs):

  • CO1: Structure Java class frameworks utilizing encapsulation and polymorphism.
  • CO2: Deploy safe concurrent multi-threaded execution codes.
  • CO3: Integrate client socket connections to transfer serialized data streams.

Syllabus:

  • Unit I: Fundamentals of OOP, Classes, Objects, and Access Modifiers
  • Unit II: Inheritance, Interfaces, Abstract Classes, and Packages
  • Unit III: Exception Handling and Custom Exceptions Framework
  • Unit IV: Multi-Threading, Thread Lifecycle, and Concurrent Synchronization
  • Unit V: Collections Framework (ArrayList, HashMap, Sets) & File I/O
  • Unit VI: Java Socket Networking & Database Connectivity (JDBC)

Textbooks:

  • Textbook: Herbert Schildt, "Java: The Complete Reference", Oracle Press.

Academic FAQs & Rules
Syllabus & Code of Conduct
What is the attendance criteria to qualify for SPPU examinations?

As per SPPU and institute norms, a minimum of 75% attendance in both lectures and practical sessions is mandatory. Students falling below this threshold will be flagged for remedial tasks, and failure to comply may result in term work detention.

Can I get an extension on assignment deadlines?

Extensions are granted only in cases of documented medical emergency or official college representation. You must submit a request letter to Mrs. Sunita Patil prior to the due date. Late submissions without approvals will face a 20% mark deduction penalty per day.

How are laboratory term-work marks calculated?

Term work marks (typically out of 25 or 50) are continuous assessments. They map: 40% on timely lab code validation, 30% on write-up completeness, 20% on weekly viva readiness, and 10% on attendance records.

What is the exam pattern for the Practical and Viva-Voce?

Practical exams consist of a 1-hour coding session based on random slips containing course assignments. This is followed by a 10-minute viva session by an external examiner covering theory definitions, code modifications, and course objectives.
